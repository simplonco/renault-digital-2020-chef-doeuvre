## Installation

Choose either "client-angular" or "client-react", start the server (see below) and go to http://127.0.0.1:3000

```
npm install
npm start
```

## Start an Angular app

Install `npm` and `ng`, then:

```
ng init client-angular
```

## Start a React app

Install `npm`, then:

```
npx create-react-app client-react
```
